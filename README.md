# Traffic-Light-Web (OLD)
Simple web-based traffic light system using HTML, CSS, JavaScript Stack + ESP32 Arduino
(Created for Mechlab Course, June 2025)

## Features
- Website with randomly generated backdrop features
- Traffic light with 3 states to simulate an actual light
- Arduino code that links ESP32 board with 3 LEDS to simulate hardware interaction

### NOTE: 
- Make the connections on the board as per the respective GPIO pins specified in the Arduino code
- Make sure to reconfigure the hotspot name and password in the Arduino file as well
